问题一：<br />
port 命令失效
解决：<br />
下载Macport的源码，重新安装。
./configure
make 
sudo make install

问题二：<br />
jekyll serve 提示 Permission denied - /Library/Ruby/Gems/2.0.0/extensions/universal-darwin-14 。。。
解决: <br />
使用port 重新安装ruby<br />
运行 sudo jekyll serve。
